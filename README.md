# Issue Tracker - Frontend

This project serves as the frontend interface for managing and tracking project issue.

## Features

- **Create Issues**: Add new issues with title and description
- **Mark as Resolved**: Update issue status with a single click
- **Smart Filtering**: Filter issues by status (All, Open, Resolved)

## Tech Stack

- **React 19** - UI framework
- **Vite** - Build tool and dev server
- **Axios** - HTTP client for API communication
- **Tailwind CSS** - Utility-first styling
- **JavaScript (ES6+)** - Modern JavaScript features

## Installation

```bash
# Navigate to client directory
cd client

# Install dependencies
npm install

# Start development server
npm run dev

Backend Repository
This frontend connects to a .NET backend API. https://github.com/Mburg03/issue-tracker-api
```
